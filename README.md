# Console File Manager
Semestral work for the **Programming & Algorithmics** course at **CTU FIT**.

## Description
A terminal-based File Manager for UNIX systems, inspired by **Midnight Commander (mc)**.

## Features
* **Dual-Pane Interface:** Classic two-column layout for efficient file browsing.
* **RegEx Support:** Advanced file searching and filtering using Regular Expressions.
* **File Concatenation:** Built-in tool for merging files.
